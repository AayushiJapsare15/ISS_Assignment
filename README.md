##  Overview

This repository contains the **completed solutions and hands-on implementations** for the **Pre-Joining Training assignment**. The work has been implemented according to the **given syllabus and assignment guidelines** and includes the following components:

- **Core Java concepts** and object-oriented programming fundamentals  
- **Client-side programming** using **HTML, CSS, and JavaScript**  
- **Database and SQL basics**, including queries and constraints  
- **Java Web Application development** using **JSP, Servlets, Maven, and Tomcat**  
- **Basic Python programming** concepts and examples
